Lake Superior Spirits & Brews

Website Design, Server-Side Scripting

Project Overview

As an academic exercise, I designed and developed this website for Lake Superior Spirits & Brews, a company that offers a virtual tour of some of Duluth’s notable brewers and distillers. For this project I created a moodboard, designed a company logo, planned the layout, created wireframes and coded the site using Bootstrap’s fully responsive framework along with the addition of the server-side scripting language, PHP. On the backend, I used a Content Management System tool to allow the site’s dynamic content to load on-the-fly.
